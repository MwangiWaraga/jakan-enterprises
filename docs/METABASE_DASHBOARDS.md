# Suggested Metabase dashboards

## Operational dashboard

Use:

- `marts.fct_inventory_actions`
- `marts.fct_price_updates`
- `marts.dim_inventory_status`

Cards:

- OSS items to delist
- Not posted items
- Restocked items
- Price updates
- Count by action type

## Finance dashboard

Use:

- `marts.fct_kilimall_order_profit`

Cards:

- Gross sales
- Estimated net profit
- Platform fees
- Fines
- DS processing fees
- Shipping fees
- Profit by SKU
